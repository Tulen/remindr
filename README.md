# Remindr

Chrome extension that alerts users at specified times to do what they need to do. We've defaulted with a few that we personally, found quite important.

## Features

### Custom reminders
- add reminder and notes
- set time and date
- set reoccurance
- snooze 

### Future
- persisted data
- auth
- Real time mindfulness
